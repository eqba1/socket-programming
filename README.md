## A Command-Line Chat Room Application using python
- ✅ [Slide](./Socket_programming-Slide.pdf)

### Run:
> Example:

First Change the directory to `./src` then run the command below:
`Server ` : 
```bash
$ python Server.py 9998
```

`Client`: 
```bash
$ python Client.py localhost 9998
```
